# Unsupervised Learning Algorithms

This repository contains my implementations and experiments with some of the most important **Unsupervised Machine Learning** algorithms.

## Topics Covered

### 1. K-Means Clustering

K-Means is a centroid-based clustering algorithm that groups similar data points into `K` clusters.

#### What I Learned

* How K-Means works
* Choosing the number of clusters using the Elbow Method
* Cluster centroid calculation
* Data visualization of clusters
* Advantages and limitations of K-Means

#### Applications

* Customer Segmentation
* Recommendation Systems
* Image Compression
* Market Basket Analysis

---

### 2. DBSCAN (Density-Based Spatial Clustering of Applications with Noise)

DBSCAN is a density-based clustering algorithm that can discover clusters of arbitrary shapes and identify outliers.

#### What I Learned

* Core Points
* Border Points
* Noise Points
* Role of `eps` and `min_samples`
* Handling outliers effectively

#### Advantages

* No need to specify the number of clusters beforehand
* Works well with irregular cluster shapes
* Detects noise and outliers automatically

#### Applications

* Fraud Detection
* Anomaly Detection
* Geographic Data Analysis
* Network Security

---

### 3. Principal Component Analysis (PCA)

PCA is a dimensionality reduction technique used to reduce the number of features while preserving most of the information in the dataset.

#### What I Learned

* Variance and Principal Components
* Explained Variance Ratio
* Cumulative Variance
* Dimensionality Reduction
* Data Visualization in Lower Dimensions

#### Applications

* Feature Reduction
* Noise Reduction
* Data Visualization
* Faster Model Training
